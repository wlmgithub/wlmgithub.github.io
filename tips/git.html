$ git diff master...contrib

This command shows you only the work your current topic branch has introduced since its common ancestor with master. That is a very useful syntax to remember.

 https://git-scm.com/book/en/v2/Distributed-Git-Maintaining-a-Project


$ git log master..experiment

This command shows “all commits reachable from experiment that aren’t reachable from master.”

 https://git-scm.com/book/en/v2/Git-Tools-Revision-Selection#_commit_ranges 


$ git log --left-right master...experiment

This command shows what is in master or experiment but not any common references

 https://git-scm.com/book/en/v2/Git-Tools-Revision-Selection#_commit_ranges
